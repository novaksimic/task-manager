# task manger app

#### Task-manager is simple web application for managing users work tasks. This is a simple app providing data for working tasks using REST API, JSON Web Tokens, SendGrid...

## Libraries and Tools
- Node.js
- Express.js
- MongoDB
- Mongoose
- Postman
- Json WebToken
- Multer
- SandGrid
- Validator
- Bcryptjs
- Sharp

## Testing
- Jest
